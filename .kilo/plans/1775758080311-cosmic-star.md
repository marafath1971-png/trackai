## 2. Fix Scan Tab Issues

### Identified Problems
1. **Error**: "Cannot read image (this model does not support image input)"
2. Camera/image initialization can fail silently
3. No proper error handling for missing/corrupted images

### Root Causes
- Image file may be deleted before AI processes it
- No try-catch around `readAsBytes()` in `gemini_service.dart`
- Missing error UI feedback to users

### Fixes Required
1. [ ] **Add error handling** in `gemini_service.dart` around fileReading:
   ```dart
   try {
     final bytes = await imageFile.readAsBytes();
   } catch (e) {
     emitError('Unable to read image. Please try again.');
   }
   ```

2. [ ] **Validate image exists** before processing in `scan_tab.dart`:
   ```dart
   if (!await imageFile.exists()) {
     showError('Image file not found. Please try again.');
     return;
   }
   ```

3. [ ] **Add user-friendly error UI** in scan_tab.dart:
   - Show retry button on failure
   - Display clear error message

4. [ ] **Add image validity check** in AI service:
   - Validate image format (JPEG, PNG)
   - Check minimum size requirements
   - Verify image isn't corrupted

**Files:** `lib/services/gemini_service.dart`, `lib/screens/scan/scan_tab.dart`
